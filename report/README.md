#### Umlauts/Umlaute

This template supports German umlauts in the main text. However, if you need umlauts on the title page, eg. in the document title, ... and your you may have to use the native LaTeX way of writing;

| Umlaut | LaTeX way |
|--------|-----------|
| ä      | {\\"a}    |
| ö      | {\\"o}    |
| ü      | {\\"u}    |
| Ä      | {\\"A}    |
| Ö      | {\\"Ö}    |
| Ü      | {\\"Ü}    |
| ß      | {\ss}     |


### Appendix: Build notices (@harshfeudal)

You can run `.\build.bat` to build the document. If you get errors about missing LaTeX packages, you can install them using the MiKTeX console. Open the console, go to the "Packages" tab, search for the missing package name, select it and click on "Install".
